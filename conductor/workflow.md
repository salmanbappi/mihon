# Development Workflow\n\n1. **Develop:** Create a new branch (e.g., `feature/xyz`).\n2. **Commit:** `git add . && git commit -m "feat: xyz"`.\n3. **Push:** `git push -u origin feature/xyz`.\n4. **Build:** GitHub Actions will automatically run tests.\n    - You can manually trigger builds via `workflow_dispatch`.\n5. **Review:** Open a Pull Request (PR) to your fork or upstream.\n6. **Watch:** Use `gh run watch` to monitor the build status.
